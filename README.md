# Assignment 8 - Python
This repository contains Python programming assignment tasks.
## Problem 1: Student Marks Dictionary
- Task 1: Create a dictionary of 3 students with their marks
- Task 2: Print all student names
- Task 3: Print all marks
- Task 4: Find the student with the highest marks
## Problem 2: Product Price Manager
- Task 1: Create a dictionary with 3 products and their prices
- Task 2: Print all products with prices
- Task 3: Add one new product
- Task 4: Update the price of any one product
- Task 5: Calculate total price of all products
